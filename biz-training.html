员工关系管理指南

一、你的角色 二、全周期管理 三、违规处理 四、离职话术 五、证据与绩效 六、与HRBP协作 七、案例对比 八、行为红线
🎯 一、你的角色：员工关系的第一责任人
你做什么
  • 目标设定与绩效评定 —— 你是打分的人
  • 日常管理与辅导 —— 你每天面对员工
  • 过程证据收集 —— 你沉淀第一手素材
  • 离职首次沟通 —— 由你先开口
HRBP 做什么（为你托底）
  • 法律风险把控 —— 告诉你什么能做、什么不能做
  • 流程指导 —— 告诉你每一步怎么做
  • 协商谈判介入 —— 你谈不拢时接手
  • SSC 对接 —— 发法律文书（仲裁由SSC负责）
🔑
核心原则：你是员工体验的决定者，HRBP 是你的专业后盾。发现问题第一时间同步，不要独自扛到最后一刻。
🗓 二、员工全生命周期管理

从入职前到转正后，每个阶段你该做什么

📥 阶段一：入职前 —— 风险防控
你要做
精准描绘用人画像，明确岗位说明书，按需设计测试题目。发现问题及时与招聘HR沟通，遵循"谁发现谁提出"原则。
HRBP做
学信网验证学历、调取任职资格资料、查看社保记录、对有疑问者增加背调。
🔄 阶段二：试用期 —— 最关键的风险窗口
⚠
试用期是最容易合法优化的阶段！务必在入职时就与员工圈定考核目标并双方签字确认，否则后期无法证明"不符合录用条件"。
你要做
①目标设定：入职第一周圈定试用期考核目标及要求，双方确认并签字
②导师安排：安排带教导师帮助融入，提供辅导资源
③动态关注：通过正式/非正式沟通关注员工思想和生活动态
④转正评估：客观评估，原则上不可随意延长试用期
⑤不胜任处理：客观评价绩效→制定PIP→安排辅导→收集证据→协商处理，必要时邀请HRBP介入
📈 阶段三：转正后 —— 持续管理与风险预警
你要做
①目标校准：明确转正后阶段性目标和能力短板
②绩效目标：按岗位要求制定月度考核目标
③培训辅导：交付未达预期时及时提供培训和辅导
④动态关注：关注员工思想和生活动态
⑤风险应对三步骤：PIP → 培训带教 → 优先调岗 → 收集证据 → 协商处理
🚨 三、违规处理：发现即行动
常见违规场景
  • 暴力行为、职场性骚扰
  • 违反公司制度流程
  • 言论不当致公司受损
  • 信息泄露
  • 虚假打卡、旷工
  • 弄虚作假报销
你的处理步骤
  1. 发现违规 → 第一时间正面告知员工，明确要求
  2. 收集证据 → 截图、录音、监控、书面材料
  3. 发送处罚 → 口头批评→警告→通报批评→记过，逐级递进
  4. 同步HRBP → 及时上报，协商后续处理方案
  5. 关注团队 → 控制影响面，安抚团队情绪
⚠
不要：发现违规后先"忍一忍"再看情况；口头警告不留书面记录；私下解决不通知HRBP。这些都可能导致后续无法形成有效证据链。
💬 四、离职沟通话术指引

你是第一个开口的人——这些话术帮你顺利开场

场景一 人才盘点离职（789格子）

  1. 先了解近期工作情况，建立对话基础
  2. 告知盘点结果及指标定义，用数据说话
  3. 暗示公司可能的淘汰动作，给员工心理缓冲
  4. 引导员工考虑主动规划职业发展

员工同意：告知 HRBP 发起离职申请
不同意且要求赔偿：转交 HRBP 进一步沟通

场景二 编制取消离职

  1. 以部门业务调整和组织绩效发展为切入点
  2. 了解员工工作饱和度等情况
  3. 表达公司愿意提供转岗或外部推荐支持
  4. 引导员工寻找更好发展机会

协商成功：按公司政策补偿
协商不成：转交 HRBP 处理

🔑
沟通四原则：①以"公司发展需要"而非"员工不好"为基调 ②不承诺具体补偿金额，统一"按公司政策执行" ③给员工24-48小时冷静期 ④全程至少两人在场，做好记录
📎 五、证据收集与绩效评语实操

你日常沉淀的证据，决定了优化时能不能站住脚

你需要收集的日常证据清单

✅ 随时收集

  • 工作布置与交付记录（飞书任务截图）
  • 客户/同事投诉（飞书聊天/微信截图）
  • 会议签到与培训记录
  • 一对一沟通纪要（会后邮件确认）

❌ 重大事件必留

  • 工作重大失误（量化损失金额）
  • 违纪行为（时间/地点/在场人）
  • 拒签文件（保留送达记录）
  • 绩效面谈（双方签字或邮件确认）

✅ 绩效评语这样写

  • "截至X月X日，XX报告未提交"
  • "当月XX指标完成XX%，低于目标XX%"
  • "收到客户/同事X次投诉"
  • 得分与评语严格匹配

❌ 千万不要这样写

  • "员工态度差""有待提升""表现一般"
  • 绩效打低分但评语写"表现尚可"
  • "让人失望""不配合工作"
  • 缺乏任何具体数据和实例
💡
给管理者的真心话：不要因为"不好意思"而写出"该员工整体表现尚可，但仍需努力"——这种评语与低分不匹配，在仲裁中会被认为考核不公正，导致败诉。 评分与评语必须严格对应。
🤝 六、你与 HRBP 的协作流程

什么时候找 HRBP、带什么信息、期望什么结果

发现问题 → 第一时间同步

员工绩效下滑、违纪行为、团队负面反馈 —— 立刻告知 HRBP，不要等"再看看"

日常管理 → 持续留痕

考核评分、PIP制定、辅导记录 —— 主动沉淀证据，HRBP 定期审核有效性

决定优化 → 联合方案

你提出优化诉求，HRBP 评估法律风险和适用类型，共同确定策略

首次沟通 → 你来说

按话术指引先沟通，让员工从直属上级口中听到消息（而非突然从HR处得知）

协商不成 → HRBP接手

员工不同意或要求赔偿时，立刻转交 HRBP，不要自行承诺任何条件

离职执行 → HRBP/SSC主导

后续法律文书发送、仲裁应对由 SSC 负责，HRBP 协调流程，你配合提供证据材料

⚖ 七、案例对比：做对了什么、做错了什么

同样的场景，不同的操作，结果天差地别 — 你的每一步动作都算数

📋
案例一背景：员工张三，试用期3个月，多次未按时完成任务且交付成果错误明显（3次书面指正无改进），3项量化考核均未达标（56分/合格70分），缺少岗位所需协作能力导致两个项目受阻。
❌ 败诉案例 —— 业务管理者错在哪里
  • 评语太虚："员工态度不好""有待提升"，没有一条量化数据
  • 流程倒置：没做完绩效评定就让员工走了，员工不确认结果
  • 未书面告知：没当面/邮件告知绩效结果及不符合录用的事实
  • 无过程管理：没制定试用期考核计划，没做定期回顾
⚠
败诉后果：仲裁裁定违法解除，赔偿2N — 就因为业务管理者日常没留证据
✅ 胜诉案例 —— 业务管理者做对了什么
  • 过程管理到位：每项指标打低分，评语含量化数据（"截至X日XX报告未提交""XX指标仅完成XX%"）
  • 得分与评语一致：1.66分配"低于预期"，逻辑自洽
  • 证据先行：离职面谈时当面告知绩效结果+不符合录用的事实依据，固化证据后才发解除通知
  • 全程留痕：试用期考核计划、定期回顾记录、3次书面指正均有存档
✓
胜诉关键：业务管理者日常做对了"量化+留痕+流程"，HRBP就有了打赢仲裁的底气
💡
给你的一句话总结：你不是HR，但你每天都在决定公司打官司时能不能赢。绩效评语里多写一句量化数据，仲裁时就多一分胜算。
🚫 八、行为红线：绝对不要做
  1. 不要口头承诺补偿金额。所有补偿条件由 HRBP 统一口径，你说"按公司政策"即可。
  2. 不要单方面通知解除。所有正式解除必须经 HRBP 和法律流程，你不可以直接让员工"明天别来了"。
  3. 不要销毁或隐瞒证据。无论对员工评价如何，客观保留所有记录。伪造或销毁证据会导致公司在仲裁中处于绝对劣势。
  4. 不要在公开场合进行离职面谈。尊重员工尊严，安排私密空间，避免当众羞辱。
  5. 不要在团队中散播优化信息。所有沟通严格保密，由 HRBP 统一管理信息发布节奏。
  6. 不要私下了结违规事件。发现严重违规必须上报 HRBP，私下处理可能给公司带来更大法律风险。
